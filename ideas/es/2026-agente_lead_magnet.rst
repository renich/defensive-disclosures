Resumen Técnico
===============
Sistema de "Imán de Prospectos" (Lead Magnet) automatizado diseñado para capturar y calificar clientes potenciales mediante la entrega inmediata de valor técnico personalizado. El sistema cuenta con un "Motor de Valor" que realiza una tarea específica basada en datos para el usuario (ej. una auditoría SEO de un sitio web, un cálculo de ahorro de impuestos o una estimación de valoración de la propiedad) a cambio de su información de contacto. El motor utiliza la ingesta de datos en tiempo real y Modelos de Lenguaje Extensos (LLM) para generar un "Informe Personalizado" de alta calidad que se entrega al instante. El sistema cuenta con un módulo de "Calificación de Prospectos" integrado que analiza los datos de entrada del usuario para asignar una puntuación de "Prioridad de Ventas", lo que permite a las empresas identificar prospectos de alto valor y automatizar sus flujos de trabajo de seguimiento a través de la integración con CRM.

Campo Técnico
=============
This disclosure relates to the field of MarTech, automated data analysis, and lead generation systems.

Antecedentes y Problema Técnico
===============================
Traditional "Lead Magnets" (e.g., generic eBooks or Whitepapers) have decreasing conversion rates because they require the user to wait for value. Modern consumers expect "Instant Gratification." A business that provides a specific, useful calculation or audit for a user's *exact* situation is much more likely to capture their interest. However, manually creating these reports is impossible at scale. There is a need for a system that automates the "Value-for-Data" exchange by performing complex analysis in seconds.
